# StellarSat SmokeWatch – Satellite Based Smoke Detection System

## Overview

StellarSat SmokeWatch is an ESP32-based atmospheric smoke detection and monitoring system designed for real-time air quality analysis.

The system integrates multiple environmental sensors for smoke detection and atmospheric monitoring.

## Features

* Real-time smoke detection
* ESP32-based embedded system
* Environmental monitoring
* Custom PCB design in EasyEDA
* Sensor data visualization
* Simulated satellite telemetry communication

## Hardware Components

* ESP32
* MQ-2 Gas Sensor
* GP2Y1010AU0F Dust Sensor
* BME680 Environmental Sensor

## Communication Interfaces

* UART
* I2C
* GPIO
